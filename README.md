## General Dependenices
  * Python(In general [eel](https://github.com/ChrisKnott/Eel))
  * JS

 ## Specific Dependencies
  * weather module:
    * pyowm
   
  * timezone and geo modules:
    * geocoder
    * pytz
    * timezonefinder
    * datetime

 * translation module: 
    * translate

    ## Usage
      * Clone the repo, and then
      ```sh
      $ cd simple-weather-app
      $ pip install -r requirements.txt
      $ python main.py
      ```
